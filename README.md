### 🎓 Final Year Project — Bill Scanning Based Reward & Intelligence System

A production-grade multimodal AI platform built by **Team ARAJ** as the BITS Pilani capstone. A multilayer OCR pipeline — a custom ML model trained on the CORD and SROIE datasets plus 200+ manually annotated real-world receipts — hands off to a Python Flask microservice calling the Google Gemini multimodal API for structured data extraction, powering a dynamic reward intelligence engine.

**Stack:** React.js (Vite) · Node.js/Express · Python/Flask · MongoDB/Firebase · JWT Auth · Google Gemini API

🔗 [Project Repo](https://github.com/ashfaaqkt/Bill-Scanning-Based-Reward-Intelligence-System-study-project-bits-poc-phase-3-Team-ARAJ)
